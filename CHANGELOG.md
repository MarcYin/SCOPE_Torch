# Changelog

This project uses automated GitHub release notes as the canonical changelog.

- Draft release notes are maintained by `.github/workflows/release-drafter.yml`
- Tagged releases are published by `.github/workflows/release.yml`
- Distribution artifacts are accompanied by GitHub artifact attestations generated during the release workflow

For installed package versions and release assets, see:

- GitHub Releases for this repository
- the `SCOPE-RTM` project on PyPI
